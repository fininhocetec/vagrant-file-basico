Vagrantfile - Básico
========================

Um Vagrantfile básico com duas maquinas.


Requisitos
------------

Instalação do VirtualBox como  provisor e Vagrant.


Dependencies
------------

None

Rodando o Ambiente
-------------------

Tenha certeza de star na pasta onde encontra o Vagrantfile 

$ vagrant up

$ vagrant ssh server1
  exit

$ vagrant ssh server2
  exit 

$ vagrante status

$ vagrant global-status

$ vagrant halt

$ vagrant destroy
